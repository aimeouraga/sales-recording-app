## 1. Description
This desktop application allows merchants to record sales easily.
Each sale includes product name, quantity, unit price, and total price.

## 2. System Requirements
- Windows / macOS / Linux
- Internet required

## 3. Running the Application Locally
- Clone the repository

    git clone <repository_url>

- Navigate to the project directory

    cd <project_directory>

- Install the dependencies

    pip install -r requirements.txt

- Start the Streamlit application

    streamlit run ðŸš€_Application.py

Once the application starts, a local URL will be displayed in the terminal (usually http://localhost:8501).

Open this link in your web browser to access the application.

## 4. How to Record a Sale
1. Go to the "Sales" page
2. Enter product name
3. Enter quantity
4. Enter unit price
5. Click "Add Sale"

## 5. Automatic Features
- Total price is calculated automatically
- Sales are saved automatically
- Data persists after closing the app
